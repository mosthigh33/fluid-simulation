2. Python-Programm für Kaleidoskop-Videos

python
import cv2
import numpy as np
import ffmpeg
from tkinter import filedialog, Tk

# Funktion, um ein Kaleidoskop-Video zu erstellen
def kaleidoscope_effect(frame, n_slices=6):
    """Fügt Kaleidoskopeffekt auf ein Frame an"""
    height, width, _ = frame.shape
    center = (width // 2, height // 2)
    angle_step = 360 // n_slices
    result = np.zeros_like(frame)

    for i in range(n_slices):
        angle = i * angle_step
        M = cv2.getRotationMatrix2D(center, angle, 1)
        rotated = cv2.warpAffine(frame, M, (width, height))
        mask = np.zeros_like(frame)
        cv2.fillPoly(mask, [np.array([[0, 0], [width, 0], [width, height], [0, height]])], (255, 255, 255))
        mask = cv2.warpAffine(mask, M, (width, height))
        mask = mask > 0
        result[mask] = rotated[mask]
    
    return result

# Funktion, um das Video zu erstellen und zu speichern
def create_kaleidoscope_video(input_video_path, output_video_path, n_slices=6, resolution=(3840, 2160)):
    """Erstellt ein Kaleidoskop-Video und speichert es im angegebenen Pfad"""
    cap = cv2.VideoCapture(input_video_path)
    
    # Prüfen, ob das Video geöffnet werden konnte
    if not cap.isOpened():
        print("Fehler beim Öffnen des Videos!")
        return

    # Video-Writer initialisieren
    fourcc = cv2.VideoWriter_fourcc(*'XVID')
    out = cv2.VideoWriter(output_video_path, fourcc, 30.0, resolution)

    while cap.isOpened():
        ret, frame = cap.read()
        if not ret:
            break

        # Kaleidoskop-Effekt anwenden
        kaleidoscoped_frame = kaleidoscope_effect(frame, n_slices)
        
        # Frame ins Video schreiben
        out.write(kaleidoscoped_frame)

    cap.release()
    out.release()
    print(f"Kaleidoskop-Video wurde erfolgreich erstellt: {output_video_path}")

# Hauptfunktion zur Auswahl des Videos und Festlegung der Ausgabedatei
def main():
    root = Tk()
    root.withdraw()  # Versteckt das Tkinter-Hauptfenster

    input_video_path = filedialog.askopenfilename(title="Wähle das Eingabevideo")
    output_video_path = filedialog.asksaveasfilename(defaultextension=".mp4", title="Wähle Ausgabedatei")

    # Erstellen des Kaleidoskop-Videos mit 4K-Auflösung
    create_kaleidoscope_video(input_video_path, output_video_path, n_slices=8, resolution=(3840, 2160))

if __name__ == "__main__":
    main()

Erklärung der Funktionen:

  • kaleidoscope_effect: Diese Funktion nimmt ein einzelnes Video-Frame und fügt den Kaleidoskop-Effekt hinzu. Sie teilt das Bild in n_slices Teile (z.B. 6 oder 8) und spiegelt oder dreht diese Teile entsprechend, um den Kaleidoskop-Effekt zu erzielen.

  • create_kaleidoscope_video: Diese Funktion liest das Eingangsvideo, wendet den Kaleidoskop-Effekt auf jedes Frame an und speichert das bearbeitete Video in einer 4K-Auflösung (3840 x 2160).

  • main: Das Skript öffnet eine einfache Benutzeroberfläche (mit Tkinter), mit der der Benutzer ein Eingabevideo auswählen und die Ausgabedatei speichern kann. Das Video wird mit einer Auflösung von 4K exportiert.

Technische Lösung:

1. Frontend: HTML + JavaScript (für den Kaleidoskop-Effekt)

Wir können das Kaleidoskop mit HTML5 <canvas> und JavaScript erstellen. Der Benutzer lädt das Video hoch, und das Video wird in ein Canvas gezeichnet. Der Kaleidoskop-Effekt wird dann im Canvas auf jedes Frame angewendet.

HTML + JavaScript Code (Frontend):

html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Kaleidoskop Video Ersteller</title>
    <style>
        body { font-family: Arial, sans-serif; }
        .container { max-width: 1200px; margin: 0 auto; text-align: center; padding-top: 30px; }
        canvas { border: 1px solid #ddd; }
    </style>
</head>
<body>
    <div class="container">
        <h1>Kaleidoskop Video Ersteller</h1>
        <input type="file" id="videoInput" accept="video/*"><br><br>
        <label for="slices">Anzahl der Kaleidoskop-Schnitte:</label>
        <input type="number" id="slices" value="6" min="2" max="12"><br><br>
        <button onclick="startKaleidoscope()">Video Starten</button><br><br>
        <video id="videoElement" width="640" controls style="display:none;"></video><br><br>
        <canvas id="canvas" width="640" height="360"></canvas>
    </div>

    <script>
        const videoElement = document.getElementById('videoElement');
        const canvas = document.getElementById('canvas');
        const ctx = canvas.getContext('2d');
        let video = null;

        // Kaleidoskop-Effekt auf das Video anwenden
        function kaleidoscopeEffect(frame, slices) {
            const width = frame.width;
            const height = frame.height;
            const center = { x: width / 2, y: height / 2 };
            const angleStep = 360 / slices;
            ctx.clearRect(0, 0, width, height);
            for (let i = 0; i < slices; i++) {
                const angle = i * angleStep;
                ctx.save();
                ctx.translate(center.x, center.y);
                ctx.rotate((angle * Math.PI) / 180);
                ctx.drawImage(frame, -center.x, -center.y, width, height);
                ctx.restore();
            }
        }

        // Startet das Video mit Kaleidoskop-Effekt
        function startKaleidoscope() {
            const slices = parseInt(document.getElementById('slices').value);

            if (video) {
                video.pause();
            }

            const videoFile = document.getElementById('videoInput').files[0];
            if (videoFile) {
                const videoURL = URL.createObjectURL(videoFile);
                videoElement.src = videoURL;
                videoElement.style.display = 'block';
                videoElement.play();
                processVideo(videoElement, slices);
            }
        }

        // Frames des Videos auf das Canvas anwenden
        function processVideo(video, slices) {
            const drawFrame = () => {
                if (video.paused || video.ended) return;
                ctx.drawImage(video, 0, 0, canvas.width, canvas.height);
                kaleidoscopeEffect(canvas, slices);
                requestAnimationFrame(drawFrame);
            };
            drawFrame();
        }

    </script>
</body>
</html>

Erklärung:

  1. HTML-Struktur: Das Interface ermöglicht es dem Benutzer, ein Video hochzuladen, die Anzahl der Kaleidoskop-Schnitte auszuwählen und dann das Video mit dem Effekt zu starten.
  2. JavaScript:
    • Kaleidoskop-Effekt: Der Effekt wird auf das canvas angewendet, indem das Video-Frame gezeichnet und anschließend rotiert und geschnitten wird. Dabei wird das Canvas als Zwischenablage genutzt, um den Effekt in Echtzeit anzuwenden.
    • Starten des Videos: Der Benutzer lädt ein Video hoch, und das Video wird auf das Canvas angewendet. Mit requestAnimationFrame() wird jedes Frame des Videos verarbeitet und der Kaleidoskop-Effekt in Echtzeit angewendet.

2. Backend (Server für Videoverarbeitung)

Für komplexe Videoverarbeitung (wie das Erstellen eines vollständigen Videos im Kaleidoskop-Stil und das Exportieren in 4K) benötigen Sie möglicherweise ein Backend. Ein Server könnte mit Node.js oder einer anderen Backend-Technologie FFmpeg nutzen, um Videos zu bearbeiten und zu exportieren.

Beispiel eines einfachen Node.js-Backends (mit FFmpeg):

  1. Backend-Server (Node.js + FFmpeg)

Installieren Sie zunächst Node.js und die erforderlichen Pakete:

bash
npm init -y
npm install express multer ffmpeg-static

Erstellen Sie dann eine einfache Express-App für den Video-Upload und die Verarbeitung:

javascript
const express = require('express');
const multer = require('multer');
const ffmpeg = require('fluent-ffmpeg');
const path = require('path');
const app = express();
const upload = multer({ dest: 'uploads/' });

app.use(express.static('public'));

// Endpunkt für den Video-Upload
app.post('/upload', upload.single('video'), (req, res) => {
    const videoPath = req.file.path;
    const outputPath = path.join('output', 'kaleidoscope_video.mp4');

    // FFmpeg Videoverarbeitung (Kaleidoskop-Effekt)
    ffmpeg(videoPath)
        .output(outputPath)
        .videoFilters('split=3[v0][v1][v2];[v0]rotate=45*PI/180[v0out];[v1]rotate=90*PI/180[v1out];[v2]rotate=135*PI/180[v2out]')
        .on('end', () => {
            console.log('Video verarbeitet');
            res.sendFile(path.resolve(outputPath));
        })
        .run();
});

// Server starten
app.listen(3000, () => {
    console.log('Server läuft auf http://localhost:3000');
});

Dieses Backend ermöglicht es, ein Video hochzuladen, und es wird auf dem Server mithilfe von FFmpeg bearbeitet und zurückgegeben.

3. Fazit

Für die Browser-basierte Lösung benötigen Sie HTML5, JavaScript und das <canvas>-Tag, um in Echtzeit den Kaleidoskop-Effekt anzuwenden. Für eine komplexere Videoerstellung und -bearbeitung (insbesondere das Rendern von 4K-Videos oder die Verarbeitung großer Video-Dateien) ist ein Backend erforderlich, um FFmpeg auf dem Server auszuführen.

Weitere Überlegungen:

  • Leistung: Die Videoverarbeitung im Browser könnte bei großen Dateien oder komplexen Effekten begrenzt sein. Ein Backend mit Server-Power wäre in diesem Fall vorzuziehen.
  • Export von 4K-Videos: Falls das Video vollständig exportiert werden soll, sollte der Server eine höhere Auflösung wie 4K unterstützen und die Dateien an den Benutzer zurücksenden.
